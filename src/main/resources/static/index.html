사용 방법
1

구글폼 설문 만들기

아래의 구글폼 설문 문항 구성 양식과 동일하게 오피스아워 담당자가 구글폼을 제작한다.

2

응답 수집

구글폼 링크를 공유 후, 응답을 수집한다.

3

CSV 다운로드

구글폼 → 응답 탭 → 스프레드시트 아이콘 → Google Sheets 열기
→ 파일 → 다운로드 → Microsoft Excel (.xlsx)

4

직책별 배정 횟수 설정 & 운영 슬롯 설정
